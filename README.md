# Paula La Rosa - Portfolio

Creative Frontend Developer Portfolio built with React, TypeScript, and Tailwind CSS.

## Project info

This project showcases the portfolio of Paula La Rosa, developed with modern frontend technologies and an emphasis on design and performance.

**Main Case**: [Khaos Kontrol](https://khaoskontrol.com.br)

## Technologies


- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase
- Framer Motion

## Getting Started

### Prerequisites

- Node.js & npm

### Installation

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm i

# Start the development server
npm run dev
```

## Deployment

Build for production:

```sh
npm run build
```
